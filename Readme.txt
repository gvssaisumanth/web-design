# web-design
Task: To create a multiple page website using HTML5 tags and CSS

Implementation: 

I have designed a coffee website with 5 pages
Home page, Order, Services, Media and article page

Tags Used: 

<html>, <head>, <link>, <title>, <body>, <header>, <div>,
<h3>, <nav>, <li>, <a>, <main>, <section>, <header>
<video>, <p>, <form>, <img>, <footer>,<article>,<<video>,<source>,<br>


<html> - It is a container of all HTML elements in the document
<head> - It is a container of metadata
<link> - it is used to link external stylesheets or add favicon
<title> - It defines the title of the document
<body> - It defines the document body
<header> - It represents a container for introduction
<div> - It defines a section in a HTML document
<h3> - Used for headings
<nav> - Defines set of navigation links
<li> - ordered list
<a> - Used to define a hyperlink to link different HTML pages
<main> - Specifies main content of the document
<section> - Defines a section ina HTML document
<video> - Used to insert a video in a HTML page
<p> - paragraph
<img> - used to insert image
<footer> - used to add footer to the document
<article> -used to specify independent, self-contained content.
<source> - tag is used to specify multiple media resources for media elements,
